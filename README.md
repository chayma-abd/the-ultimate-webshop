Mijn webshop project;
Ik heb een webshop gemaakt voor Zara Cosmetics waar je parfums kunt kopen. Het heeft een mooie winkelkant voor klanten en een apart beheergedeelte voor de shopbeheerder.
Wat ik heb gemaakt:
Voor klanten: Een homepage met alle parfums en aparte pagina's voor elk product
Voor de shopbeheerder: Een speciaal admin paneel om producten te beheren
Hoe het werkt: Gebruikt HTML en CSS, werkt op telefoon en computer
Bronnen die ik heb gebruikt
Ik heb deze bronnen gebruikt om mijn project te maken:
Canvas video's - De instructievideo's die op Canvas stonden om HTML en CSS te leren
PowerPoint - De lesstof en presentaties van de docent
AI assistentie - Voor sommige stukken code (zoals de admin panel) heb ik hulp gekregen van een AI
Canva - Voor het maken en bewerken van afbeeldingen voor mijn webshop
Hoe je het bekijkt
Open de map en dubbelklik op index.html voor de webshop
Voor het admin paneel: ga naar de admin map en open index.html
Inloggen als beheerder: gebruik admin en wachtwoord admin123
